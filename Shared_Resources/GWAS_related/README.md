## Goal: A place to store GATK-related notes

- [GATK Cloud Resources on Terra](https://gatk.broadinstitute.org/hc/en-us/articles/4524513257499-GATK-on-the-cloud-with-Terra)  
  Overview of how to use GATK workflows and tools on the Terra cloud platform.

- [GATK-SV Website](https://broadinstitute.github.io/gatk-sv/)  
  Official website for the GATK-SV pipeline, including workflow documentation and usage notes for structural variant analysis.

- [Somatic Variant Calling with Mutect2 Notebook](https://app.terra.bio/#workspaces/help-gatk/Somatic-SNVs-Indels-GATK4/analysis/launch/Somatic-Mutect2-Tutorial.ipynb)  
  Terra notebook tutorial for somatic SNV and indel calling with Mutect2, especially useful for tumor-normal pair analysis.

## Online Tutorials

- [GWASTutorial](https://cloufield.github.io/GWASTutorial/)  
  A detailed introductory tutorial series on GWAS and related analyses, hosted by the Laboratory of Complex Trait Genomics at the University of Tokyo.
