# Documentation & Useful Links

## Genome Browsers & Visualization Tools
- [UCSC Genome Browser](https://genome.ucsc.edu/) - Comprehensive genome annotation browser
- [IGV (Integrative Genomics Viewer)](https://igv.org/) - Desktop visualization tool for genomic data
- [Ensembl Genome Browser](https://www.ensembl.org/) - Genome annotation and comparative genomics

## Data Repositories & Portals
- [GTEx Portal](https://gtexportal.org/) - Gene expression across human tissues
- [GDC Data Portal (TCGA)](https://portal.gdc.cancer.gov/) - Cancer genomics data
- [dbGaP](https://www.ncbi.nlm.nih.gov/gap/) - Database of Genotypes and Phenotypes
- [NCBI GEO](https://www.ncbi.nlm.nih.gov/geo/) - Gene Expression Omnibus

## Databases
- [ClinVar](https://www.ncbi.nlm.nih.gov/clinvar/) - Clinical variant interpretations
- [gnomAD](https://gnomad.broadinstitute.org/) - Genome aggregation database
- [COSMIC](https://cancer.sanger.ac.uk/cosmic) - Catalogue of somatic mutations in cancer
- [dbSNP](https://www.ncbi.nlm.nih.gov/snp/) - Single nucleotide polymorphism database

## Analysis Tools & Platforms
- [Galaxy](https://usegalaxy.org/) - Web-based analysis platform
- [Bioconductor](https://www.bioconductor.org/) - R packages for genomics
- [GATK](https://gatk.broadinstitute.org/) - Genome Analysis Toolkit

## Reference Genomes
- [GENCODE](https://www.gencodegenes.org/) - Human and mouse gene annotations
- [RefSeq](https://www.ncbi.nlm.nih.gov/refseq/) - NCBI Reference Sequence Database
